This is currently a simple stock market webapp which allows registered users access to US stock market information (data and news) usig a third party API - Tiingo.
Regitered users can sign in and deposit money(it's currently a fixed deposit amount but that will change with later commits and I havent tested with real Cards) into their accounts which is automatically reflected on their dashboards using the Braintree Sandbox API. 
Users also have access to a blog site where they can read news about the latest happenings in the stock market.
